# AUTONOMOUS_STRATEGY — Дворівнева автономна архітектура

## Концепція

Система побудована на принципі **"Учитель — Учень"**: потужний вузол навчає модель, легкий вузол виконує.

---

## Рівень 1: Master Node (Tier 1)

### Призначення
- Навчання ML-моделей
- Дистиляція знань
- Зберігання бази сигнатур
- Координація Edge-вузлів

### Апаратне забезпечення
- Сервер з 4× GPU (A100 або RTX 4090)
- 128+ GB RAM
- NVMe storage 2TB+

### Підключення
Через зашифрований канал (WireGuard) до Base Station.

---

## Рівень 2: Edge Node (Tier 2)

### Призначення
- Збір даних з SFP
- Швидкий FFT-аналіз
- Класифікація в реальному часі
- GenAI-сповіщення

### Апаратне забезпечення
- Raspberry Pi 5 (8GB)
- Або Nvidia Jetson Orin
- Або міні-PC (Intel N100)

### Місце
Окоп / позиція / техніка

---

## Автономний ланцюжок

```
[ Волокно ] → [ SFP ] → [ Edge Node ]
                              │
                              ▼
                     ┌─────────────────┐
                     │  Edge Inference │  ← локально
                     │  ML: 50-500MB │
                     └─────────────────┘
                              │
                  [ WireGuard тунель ]
                              │
                              ▼
                     ┌─────────────────┐
                     │  Master Node   │  ← навчання
                     │  (тиха зона)  │
                     └─────────────────┘
```

---

## Режими роботи

### Mode A: Повна автономія
```
Master Node: ОФЛАЙН
Edge Node: ЛОКАЛЬНИЙ
  → евристична класифікація
  → буфер даних
  → ручний аналіз
```

### Mode B: Коллаборація
```
Master Node: ОНЛАЙН
Edge Node: ПІДКЛЮЧЕНИЙ
  → передача векторів
  → оновлення ваг
  → спільний інтелект
```

### Mode C: Навчання
```
Master Node: НАВЧАННЯ
Edge Node: ЗБІР
  → збір сирих даних
  → розмітка
  → дистиляція
```

## Безпека

- Жодного радіовипромінювання
- шифрування ChaCha20-Poly1305
- локальні дані не йдуть в хмару
- фізична ізоляція
