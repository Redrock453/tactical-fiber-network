# Фізична модель сенсорної мережі SpiderLink

## 1. Розподілене акустичне зондування (DAS)

В основі лежить явище **релеївського розсіювання**. Кожна мікронеоднорідність всередині кварцевого скла відбиває частину світла назад. При механічному впливі (кроки, техніка) змінюється відстань між цими точками.

### Фазова модуляція

Будь-яке стиснення або розтягнення волокна на величину $\epsilon$ призводить до зміни фази світлової хвилі $\Delta\phi$:

$$\Delta\phi = \frac{4\pi n L}{\lambda} \left( 1 - \frac{n^2}{2} [p_{12} - \nu(p_{11} + p_{12})] \right) \epsilon$$

Де:
- $p_{11}, p_{12}$ — фото-пружні коефіцієнти
- $\nu$ — коефіцієнт Пуассона
- $\lambda$ — робоча довжина хвилі (1550 нм)

### Практичні значення

| Параметр | Значення |
|----------|----------|
| Чутливість Δφ | 10⁻⁶ рад |
| Довжина хвилі λ | 1550 нм |
| Показник заломлення n | 1.47 |

## 2. Електрооптична детекція (Ефект Керра)

При впливі потужного електромагнітного поля (сигнал РЕБ) на волокно, показник заломлення скла $n$ змінюється пропорційно квадрату напруженості поля $E$:

$$n(E) = n_0 + n_2 |E|^2$

Це дозволяє використовувати волокно як **розподілену антену**. Навіть не маючи прямого електричного контакту, ми фіксуємо "тремтіння" оптичного сигналу, спричинене роботою ворожих радіоелектронних засобів.

### Коефіцієнт Керра

Для кварцевого скла:
- $n_2 = 3.2 \times 10^{-20}$ м²/В²

### Приклад розрахунку

РЕБ потужністю 1 кВт на відстані 5 метрів:

```
E ≈ 170 В/м
Δn = 3.2 × 10⁻²⁰ × 170² ≈ 9.2 × 10⁻¹²
Δφ ≈ 0.037 рад (вимірювально!)
```
