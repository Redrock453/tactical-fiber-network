# TFN — Трофейная разведка (Trophy Intelligence)

## Концепция

**Трофейная разведка** — это подключение к **оптоволоконным линиям связи противника** для получения разведданных без обнаружения.

Если противник использует оптоволокно (а в условиях РЭБ это неизбежно), его кабель становится **вашим сенсором**.

---

## 1. Ситуация на поле боя

### 1.1 Почему у противника есть оптоволокно

В условиях мощного РЭБ:
- Радиосвязь глушится полностью
- FPV-дроны переходят на волоконное управление (кабель-катушка)
- Полевые телефоны заменяются оптическими линиями
- Появляются оптоволоконные «нитки» между позициями

**Результат:** На территории противника лежат километры оптоволокна.

### 1.2 Типы вражеских линий

| Тип | Описание | Частота встречи | Ценность |
|-----|----------|----------------|----------|
| **FPV-кабель** | Одномодовое волокно от дронов | Очень часто | Низкая (тонкое) |
| **Полевая связь** | Кабель между позициями | Часто | Высокая |
| **Штабная сеть** | Многомодовое/одномодовое к штабу | Редко | Очень высокая |
| **Гражданская инфраструктура** | Оставшиеся городские сети | В городах | Средняя |

## 2. Методы подключения

### 2.1 Обнаружение вражеского кабеля

**Признаки:**
- Чёрная/серая/жёлтая оболочка, лежит на земле или в неглубокой траншее
- Тонкий (1-3 мм) или толстый (5-10 мм) кабель
- Идёт от позиций противника
- Может быть маркирован (а может и нет)

**Инструменты поиска:**
- Визуальное наблюдение (бинокль, тепловизор ночью)
- Дневной патруль в «сёрой зоне»
- БПЛА-разведка (кабель видно с воздуха)
- Отчёты о FPV-вылетах (где падали — там кабель)

### 2.2 Подключение к обрывку

**Сценарий:** Артиллерия перебила вражеский кабель. Наш патруль находит обрывок на нейтральной полосе.

**Действия (3-5 минут):**

```
1. Найти обрывок (оба конца)
2. Очистить конец от грязи (салфетка)
3. Зачистить буфер стриппером (20 мм)
4. Протереть спиртом
5. Сколоть на 10 мм
6. Установить быстрый FC-коннектор
7. Подключить к φ-OTDR или мини-OTDR
8. Читать данные
```

**Важно:** Подключаемся к концу, который идёт **в сторону противника**. Этот конец ведёт к его позициям и будет «слышать» его активность.

### 2.3 Подключение без обрыва (TAP)

**Сценарий:** Найден целый работающий кабель противника.

**Метод 1: Изгиб (Bend Tap)**
```
1. Аккуратно изогнуть кабель под малым радиусом
2. Часть света «вытечет» через изгиб
3. Детектор рядом фиксирует утечку
4. Потери: 0.5-3 дБ (противник может заметить!)
```

**Метод 2: Splitter (Оптический ответвитель)**
```
1. Разрезать кабель (вызовет обрыв у противника!)
2. Вставить 90/10 сплиттер (90% — противнику, 10% — нам)
3. Соединить механическими сплайсами
4. Потери для противника: 0.5-1 дБ (может не заметить)
5. Время: 5-10 минут
```

**Метод 3: Расположенный рядом кабель (Coupling)**
```
1. Проложить наш кабель параллельно вражескому на 5-20 см
2. Акустическая связь между кабелями
3. Наш DAS «слышит» вибрации на вражеском кабеле
4. Бесконтактный метод — обнаружение невозможно
```

### 2.4 Таблица методов

| Метод | Обнаруживаемость | Качество данных | Время | Сложность |
|-------|------------------|-----------------|-------|-----------|
| Обрывок + OTDR | Низкая | Отличное | 3-5 мин | Низкая |
| Bend Tap | Средняя | Плохое | 30 сек | Низкая |
| Splitter TAP | Средняя | Хорошее | 10 мин | Средняя |
| Параллельный кабель | Нулевая | Среднее | 30 мин | Средняя |
| Полное подключение | Низкая | Отличное | 15 мин | Высокая |

## 3. Что можно узнать

### 3.1 DAS на вражеском кабеле

Подключив φ-OTDR к вражескому кабелю, получаем:

```
Наш конец ←──── Вражеский кабель ────→ Вражеская позиция
                    │
                    ├── 500м: Тишина (нейтральная полоса)
                    ├── 1500м: Пешеход (патруль?)
                    ├── 2000м: Техника ×2 (БТР?)
                    ├── 2800м: Копание (фортификация?)
                    ├── 3200м: Генератор 50Гц (командный пункт?)
                    ├── 3800м: Группа пехоты (15+ человек)
                    └── 4200м: Конец кабеля (или обрыв)
```

### 3.2 Перехват данных

Если враг передаёт IP-трафик по оптоволокну:
- Подключение через Splitter → сетевая карта → Wireshark
- Анализ незашифрованного трафика (если нет VPN)
- Даже при VPN — анализ метаданных (объём, время, направление)

### 3.3 Определение топологии вражеской сети

По OTDR-трассировке:
- Расстояние между узлами
- Количество соединений (сплайсов)
- Места разветвлений (сплиттеры)
- Качество линии (→ уровень подготовки противника)

### 3.4 Обнаружение штабов и КП

Признаки по DAS-данным:
- Концентрация техники
- Генератор (50 Гц сигнал)
- Большой объём IP-трафика
- Ночная активность (смена, совещания)

## 4. Симуляция трофейной разведки

### 4.1 Сценарий «Ночная операция»

```
02:00 — Разведгруппа выходит к нейтральной полосе
02:15 — Обнаружен обрывок вражеского кабеля (одномодовый, 1.8мм)
02:17 — Зачистка, скол, установка FC-коннектора
02:19 — Подключение портативного OTDR
02:20 — Начало сканирования
02:21 — Данные:
         - Кабель 4.2 км до конца
         - 2 сплайса на 800м и 2100м
         - Активность на 1800м: пешеход
         - Активность на 3500м: техника ×3
         - Активность на 3900м: генератор → вероятно КП
02:30 — Отключение, замета следов
02:45 — Возврат на позицию
03:00 — Данные переданы на Master Node для анализа
```

## 6. Контрмеры противника (и наша защита)

### 6.1 Что может сделать противник

| Действие | Сложность | Эффективность |
|----------|-----------|---------------|
| Регулярная проверка кабеля | Высокая (под огнём) | Низкая |
| OTDR-мониторинг своей линии | Средняя | Высокая (обнаружит TAP) |
| Шифрование трафика | Средняя | Блокирует перехват данных, но не DAS |
| Экранирование кабеля | Низкая | Блокирует только RF-Opto |
| Перемещение кабеля | Высокая | DAS всё равно слышит |
| Уничтожение своего кабеля при отходе | Средняя | Конечная, но мы успеем |

### 6.2 Наша стратегия

1. **Параллельный кабель** — бесконтактный, необнаружимый
2. **Подключение к обрывкам** — после артудара, противник не контролирует
3. **Быстрый TAP** — за 3-5 минут, потом уходим
4. **Анализ DAS, а не трафика** — не зависит от шифрования

## 7. Протокол операции

### 7.1 Подготовка

```
□ Разведданные о расположении вражеских позиций
□ Карта с отмеченными найденными кабелями
□ Комплект инструментов (см. п. 5)
□ Связь с Master Node (через TFN или WireGuard)
□ План отхода
□ Прикрытие (огневая группа)
```

### 7.2 Выход

```
□ Выход в тёмное время
□ Движение по заранее разведанному маршруту
□ Минимальный состав (2-3 человека)
□ Тепловизионная невидимость (накидки)
□ Радиомолчание (связь только через TFN при наличии)
```

### 7.3 Работа на точке

```
□ Обнаружение кабеля
□ Определение направления (к противнику / от противника)
□ Подготовка конца (стриппер → скол → коннектор)
□ Подключение OTDR
□ Снятие трассировки (30-60 сек)
□ DAS-сканирование (60-120 сек)
□ Отключение
□ Маскировка места работы
```

### 7.4 Возврат

```
□ Отход по плану
□ Передача данных на Master Node
□ Анализ результатов
□ Документирование (ISR-отчёт)
□ Планирование следующей операции
```

## 8. Правовые и этические аспекты

- Данная документация предназначена для **оборонительных целей**
- Все методы разведки через TFN являются **пассивными** — не создают излучения
- Использование оптоволокна противника в зоне боевых действий соответствует **законам и обычаям войны**
- В мирное время применение ограничено **законами о перехвате коммуникаций**
