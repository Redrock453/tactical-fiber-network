# TFN — Повне керівництво з розвідки (ISR Intelligence Guide)

## Огляд

Tactical Fiber Network перетворює **пасивний оптоволоконний кабель** на комплексну систему розвідки, спостереження та цілевказування (ISR — Intelligence, Surveillance, Reconnaissance).

**Принцип:** Жоден електронний прилад не випромінює радіохвилі. Вся розвідка базується на **аналізі оптичного сигналу** всередині волокна. Виявлення неможливе засобами РЕР/РТР противника.

## 2. DAS-розвідка — детально

### 2.1 Як працює φ-OTDR

Лазерний імпульс надсилається у волокно. Кожна мікронеоднорідність кварцового скла відбиває частину світла (Rayleigh backscatter). Фазочутливий приймач аналізує **зміни фази** відбитого сигналу.

```
Лазер → [импульс] → Волокно → [backscatter] → Фотоприёмник
                         ↑
                    Вибрация земли
                    (кроки, техника)
                    меняет фазу
                    отражённого света
```

### 2.2 Характеристики виявлення

| Ціль | Частота (Гц) | Амплітуда | Дальність від кабелю | Локалізація | Класифікація |
|------|-------------|-----------|---------------------|-------------|---------------|
| **Пішохід одиночний** | 1.5-3.0 | 0.15-0.35 | 3-10 м | ±2 м | 75-85% |
| **Група піхоти** | 1.0-4.0 | 0.25-0.55 | 5-20 м | ±3 м | 80-90% |
| **Мотоцикл/квадроцикл** | 10-25 | 0.35-0.60 | 10-30 м | ±3 м | 85-92% |
| **Колісна техніка (БТР, БМП)** | 8-20 | 0.40-0.70 | 15-50 м | ±5 м | 85-95% |
| **Гусенична техніка (танки)** | 2-8 | 0.60-0.95 | 20-80 м | ±5 м | 90-97% |
| **Артилерійський постріл** | 0.5-3 | 0.85-1.0 | 50-500 м | ±10 м | 95-99% |
| **Розрив снаряда** | 0.1-15 | 0.90-1.0 | 50-1000 м | ±15 м | 95-99% |
| **Копання** | 3-12 | 0.20-0.50 | 5-15 м | ±3 м | 70-85% |
| **Низьколітаючий БПЛА** | 30-100 | 0.10-0.35 | 10-50 м | ±10 м | 65-80% |
| **Гелікоптер** | 10-40 | 0.30-0.60 | 50-200 м | ±15 м | 80-90% |

### 2.3 Ефективна дальність DAS

| Параметр | Значення | Примітка |
|----------|----------|----------|
| Максимальна довжина одного сегмента | 40-50 км | З оптичним підсилювачем |
| Стандартна дальність | 10-20 км | Без підсилювачів |
| Просторова роздільна здатність | 1-5 м | Залежить від ширини імпульсу |
| Частота оновлення | 10-1000 Гц | Налаштовувана |
| Чутливість | 10⁻⁶ рад | Фазовий зсув |

### 2.4 Режими розвідки

#### Режим «Засідка» (Ambush Mode)
- Волокно прокладено вздовж ймовірного маршруту руху противника
- Система безперервно прослуховує, оператор у тилу
- Автоматичне оповіщення при виявленні техніки
- **Застосування:** Контроль доріг, стежок, мостів

#### Режим «Периметр» (Perimeter Mode)
- Замкнений контур волокна навколо позиції
- Виявлення проникнення з будь-якого напрямку
- Розрізнення: тварина (0.5-1 Гц, амплітуда < 0.1) vs піхота vs техніка
- **Застосування:** Охорона баз, складів, КП

#### Режим «Лінія фронту» (Frontline Mode)
- Волокно прокладено вздовж лінії бойового зіткнення (ЛБЗ)
- Виявлення скупчень техніки, початку наступу
- Контрбатарейна розвідка (виявлення пострілів)
- **Застосування:** Моніторинг активності на ЛБЗ

#### Режим «Стежка» (Trail Watch Mode)
- Волокно прокладено через ліс/зеленку стежкою
- Виявлення ДРГ, диверсійних груп
- Напрямок та швидкість руху
- **Застосування:** Контроль «сірої зони»

## 3. RF-Opto розвідка — пасивна детекція РЕБ та зв'язку

### 3.1 Принцип

Потужне електромагнітне поле впливає на світло у волокні через:
- **Ефект Керра** — зміна показника заломлення
- **Термооптичний ефект** — нагрівання від потужного RF
- **П'єзоелектричний ефект** — вібрація оболонки

Волокно працює як **пасивна антена завдовжки кілометри**.

### 3.2 Що можна виявити

| Джерело RF | Потужність | Частота | Дальність виявлення | Ймовірність |
|-------------|----------|---------|----------------------|-------------|
| **РЕБ-станція (Теребун/Ледяж)** | 100-5000 Вт | 100МГц-6ГГц | 5-50 м від кабелю | 90-99% |
| **Тактична радіостанція** | 1-50 Вт | 30-512 МГц | 2-30 м | 60-85% |
| **FPV-пульт керування** | 0.1-2 Вт | 2.4 ГГц | 1-15 м | 40-70% |
| **FPV-відеопередавач** | 0.025-1 Вт | 5.8 ГГц | 0.5-10 м | 30-60% |
| **Радар ППО** | 1-50 кВт | 1-18 ГГц | 50-500 м | 95-99% |
| **Стільникова вежа** | 20-200 Вт | 700МГц-3.5ГГц | 10-100 м | 80-95% |

### 3.3 Визначення місцезнаходження РЕБ

```
Волокно: ═══════════╳═══════════════════
                       ↑
                РЭБ здесь!
             (максимум фазового сдвига)

Показания по длине кабеля:
  0м    500м   1000м  1500м  2000м  2500м
  ──────┼──────┼──────┼──────┼──────┼──────
  ▁▁▁▁▁▂▃▅▇█▇▅▃▂▁▁▁▁▁▁▁▁▁▁▁▁▁▁▁▁▁▁
```

Точка максимуму фазового зсуву = проекція джерела RF на кабель.

### 3.4 Формула розрахунку дальності виявлення

Для EW-станції потужністю P (Вт) на відстані d (м):

```
E = sqrt(2 × P × 377 / (4π × d²))    [В/м]
Δn = n₂ × E²                          (n₂ = 3.2×10⁻²⁰ м²/В²)
ΔΦ = (2π/λ) × Δn × L_exposed          [рад]
```

Виявлення можливе при ΔΦ > 10⁻⁶ рад.

| P (Вт) | 1м | 5м | 10м | 50м | 100м |
|--------|-----|-----|------|------|-------|
| 1 Вт | ✅ | ✅ | ⚠️ | ❌ | ❌ |
| 10 Вт | ✅ | ✅ | ✅ | ⚠️ | ❌ |
| 100 Вт | ✅ | ✅ | ✅ | ✅ | ⚠️ |
| 1000 Вт | ✅ | ✅ | ✅ | ✅ | ✅ |
| 10 кВт | ✅ | ✅ | ✅ | ✅ | ✅ |

## 4. Контрбатарейна розвідка

### 4.1 Виявлення артилерійських позицій

DAS-система виявляє:
1. **Постріл** — імпульс у точці гармати (потужність 0.85-1.0, 0.5-3 Гц)
2. **Політ снаряда** — акустична тінь (якщо пролітає поруч)
3. **Розрив** — потужний імпульс (0.9-1.0, 0.1-15 Гц)

За різницею часу:
```
Δt = t_разрыва - t_выстрела
D = V_снаряда × Δt (приблизительно)
```

### 4.2 Точність

| Параметр | Значення |
|----------|----------|
| Виявлення пострілу | 95-99% на відстані до 2 км від кабелю |
| Локалізація гармати | ±50 м (потребує калібрування) |
| Виявлення розриву | 99% на відстані до 5 км |
| Локалізація розриву | ±15 м |
| Час виявлення | < 1 секунда |
| Передача координат | < 3 секунд |

### 4.3 Автоматичне цілевказування

```
[Обнаружен выстрел] → [Классификация: 152мм гаубица]
                     → [Расчёт координат]
                     → [Отправка на огневую позицию]
                     → [Контрбатарейный удар через 30-60 сек]
```

## 5. Моніторинг ліній постачання

### 5.1 Контроль доріг

Волокно, прокладене вздовж дороги, фіксує:
- **Напрямок руху** (доплерівський зсув)
- **Швидкість** (час проходження між точками)
- **Тип транспорту** (колісний/гусеничний)
- **Кількість одиниць** (підрахунок імпульсів)

### 5.2 Приклад звіту

```
=== TFN ISR REPORT ===
Время: 2026-04-18 03:45:12
Сегмент: Road monitoring, 1200-3500m

DETECTED:
  03:42:15 — Колёсная техника ×3, позиция 1450м, скорость ~35 км/ч, направление Ю→С
  03:43:30 — Колёсная техника ×3, позиция 2800м, скорость ~30 км/ч, направление Ю→С
  03:44:50 — Гусеничная техника ×1, позиция 1350м, скорость ~15 км/ч, направление Ю→С

ASSESSMENT: Колонна из 4+ единиц (3 колёсных + 1 гусеничная)
           Движение на север по маршруту supply-road-7
           Возможная переброска резервов

RECOMMENDATION: Наблюдение. Подготовка огневого задания.
```

### 6.2 Індикація підготовки наступу

```
Алгоритм обнаружения:

1. Увеличение активности копания в секторе (за 1-3 дня)
2. Появление техники (колёсной, потом гусеничной)
3. Рост радиопередач (RF-Opto детекция)
4. Прокладка новых кабелей противником

→ ВЫСОКАЯ ВЕРОЯТНОСТЬ ПОДГОТОВКИ НАСТУПЛЕНИЯ
```

## 8. Інтеграція з картою (Tactical Map)

### 8.1 Формат подій

```json
{
  "event_id": "evt_20260418_034512_001",
  "timestamp": "2026-04-18T03:45:12Z",
  "type": "target_detected",
  "source": "DAS",
  "position": {
    "fiber_m": 1450,
    "lat": 48.5123,
    "lon": 36.3456
  },
  "classification": {
    "target": "wheeled_vehicle",
    "confidence": 0.87,
    "count": 3,
    "speed_kmh": 35,
    "direction": "N"
  },
  "threat_level": "medium",
  "action": "track",
  "alert_sent": true
}
```

### 8.2 Візуалізація

Кожна подія відображається на тактичній карті:
- **Зелений** — тихо
- **Жовтий** — пішохід, копання
- **Помаранчевий** — колісна техніка, БПЛА
- **Червоний** — гусенична техніка, РЕБ
- **Блимаючий червоний** — артилерія, вибух

## 9. Захист від контрзаходів

### 9.1 Противник не може виявити TFN

| Метод виявлення | Ефективність проти TFN |
|-------------------|--------------------------|
| РТР (пеленгація) | ❌ Немає випромінювання |
| Тепловізор | ❌ Немає нагріву |
| Металошукач | ❌ Скло, не метал |
| Візуально | ⚠️ Тільки якщо натрапить |
| GPR (георадар) | ⚠️ Можливо на глибині < 20 см |

### 9.2 Противник може спробувати

| Дія | Ефективність | Контрзахід TFN |
|----------|---------------|---------------|
| Обрив кабелю артилерією | Частково | Mesh-резервування, автоперемикання |
| Перехоплення оптичного сигналу | Потрібен фізичний доступ | Шифрування WireGuard |
| Імітація вібрацій | Можлива дезінформація | ML-модель фільтрує |
| Виявлення лазера φ-OTDR | Ні, все світло всередині волокна | — |
