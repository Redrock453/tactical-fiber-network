# TFN — Полное руководство по разведке (ISR Intelligence Guide)

## Обзор

Tactical Fiber Network превращает **пассивный оптоволоконный кабель** в комплексную систему разведки, наблюдения и целеуказания (ISR — Intelligence, Surveillance, Reconnaissance).

**Принцип:** Ни один электронный прибор не излучает радиоволны. Вся разведка основана на **анализе оптического сигнала** внутри волокна. Обнаружение невозможно средствами РЭР/РТР противника.

## 2. DAS-разведка — подробно

### 2.1 Как работает φ-OTDR

Лазерный импульс посылается в волокно. Каждая микронеоднородность кварцевого стекла отражает часть света (Rayleigh backscatter). Фазочувствительный приёмник анализирует **изменения фазы** отражённого сигнала.

```
Лазер → [импульс] → Волокно → [backscatter] → Фотоприёмник
                        ↑
                   Вибрация земли
                   (кроки, техника)
                   меняет фазу
                   отражённого света
```

### 2.2 Характеристики обнаружения

| Цель | Частота (Гц) | Амплитуда | Дальность от кабеля | Локализация | Классификация |
|------|-------------|-----------|---------------------|-------------|---------------|
| **Пешеход одиночный** | 1.5-3.0 | 0.15-0.35 | 3-10 м | ±2 м | 75-85% |
| **Группа пехоты** | 1.0-4.0 | 0.25-0.55 | 5-20 м | ±3 м | 80-90% |
| **Мотоцикл/квадроцикл** | 10-25 | 0.35-0.60 | 10-30 м | ±3 м | 85-92% |
| **Колёсная техника (БТР, БМП)** | 8-20 | 0.40-0.70 | 15-50 м | ±5 м | 85-95% |
| **Гусеничная техника (танки)** | 2-8 | 0.60-0.95 | 20-80 м | ±5 м | 90-97% |
| **Артиллерийский выстрел** | 0.5-3 | 0.85-1.0 | 50-500 м | ±10 м | 95-99% |
| **Разрыв снаряда** | 0.1-15 | 0.90-1.0 | 50-1000 м | ±15 м | 95-99% |
| **Копание** | 3-12 | 0.20-0.50 | 5-15 м | ±3 м | 70-85% |
| **Низколетящий БПЛА** | 30-100 | 0.10-0.35 | 10-50 м | ±10 м | 65-80% |
| **Вертолёт** | 10-40 | 0.30-0.60 | 50-200 м | ±15 м | 80-90% |

### 2.3 Эффективная дальность DAS

| Параметр | Значение | Примечание |
|----------|----------|------------|
| Максимальная длина одного сегмента | 40-50 км | С оптическим усилителем |
| Стандартная дальность | 10-20 км | Без усилителей |
| Пространственное разрешение | 1-5 м | Зависит от ширины импульса |
| Частота обновления | 10-1000 Гц | Настраиваемая |
| Чувствительность | 10⁻⁶ рад | Фазовый сдвиг |

### 2.4 Режимы разведки

#### Режим «Засада» (Ambush Mode)
- Волокно проложено вдоль вероятного маршрута движения противника
- Система непрерывно слушает, оператор в тылу
- Автоматическое оповещение при обнаружении техники
- **Применение:** Контроль дорог, троп, мостов

#### Режим «Периметр» (Perimeter Mode)
- Замкнутый контур волокна вокруг позиции
- Обнаружение проникновения с любого направления
- Различение: животное (0.5-1 Гц, амплитуда < 0.1) vs пехота vs техника
- **Применение:** Охрана баз, складов, КП

#### Режим «Линия фронта» (Frontline Mode)
- Волокно проложено вдоль линии боевого соприкосновения (ЛБС)
- Обнаружение скоплений техники, начала наступления
- Контрбатарейная разведка (обнаружение выстрелов)
- **Применение:** Мониторинг активности на ЛБС

#### Режим «Тропа» (Trail Watch Mode)
- Волокно проложено через лес/зелёнку по тропе
- Обнаружение ДРГ, диверсионных групп
- Направление и скорость движения
- **Применение:** Контроль «сірої зони»

## 3. RF-Opto разведка — пассивная детекция РЭБ и связи

### 3.1 Принцип

Мощное электромагнитное поле воздействует на свет в волокне через:
- **Эффект Керра** — изменение показателя преломления
- **Термооптический эффект** — нагрев от мощного RF
- **Пьезоэлектрический эффект** — вибрация оболочки

Волокно работает как **пассивная антенна длиной километры**.

### 3.2 Что можно обнаружить

| Источник RF | Мощность | Частота | Дальность обнаружения | Вероятность |
|-------------|----------|---------|----------------------|-------------|
| **РЭБ-станция (Теребун/Ледиаж)** | 100-5000 Вт | 100МГц-6ГГц | 5-50 м от кабеля | 90-99% |
| **Тактическая радиостанция** | 1-50 Вт | 30-512 МГц | 2-30 м | 60-85% |
| **FPV-пульт управления** | 0.1-2 Вт | 2.4 ГГц | 1-15 м | 40-70% |
| **FPV-видеопередатчик** | 0.025-1 Вт | 5.8 ГГц | 0.5-10 м | 30-60% |
| **Радар ПВО** | 1-50 кВт | 1-18 ГГц | 50-500 м | 95-99% |
| **Сотовая вышка** | 20-200 Вт | 700МГц-3.5ГГц | 10-100 м | 80-95% |

### 3.3 Определение местоположения РЭБ

```
Волокно: ═══════════╳═══════════════════
                      ↑
               РЭБ здесь!
            (максимум фазового сдвига)

Показания по длине кабеля:
  0м    500м   1000м  1500м  2000м  2500м
  ──────┼──────┼──────┼──────┼──────┼──────
  ▁▁▁▁▁▂▃▅▇█▇▅▃▂▁▁▁▁▁▁▁▁▁▁▁▁▁▁▁▁▁▁
```

Точка максимума фазового сдвига = проекция источника RF на кабель.

### 3.4 Формула расчёта дальности обнаружения

Для EW-станции мощностью P (Вт) на расстоянии d (м):

```
E = sqrt(2 × P × 377 / (4π × d²))    [В/м]
Δn = n₂ × E²                          (n₂ = 3.2×10⁻²⁰ м²/В²)
ΔΦ = (2π/λ) × Δn × L_exposed          [рад]
```

Обнаружение возможно при ΔΦ > 10⁻⁶ рад.

| P (Вт) | 1м | 5м | 10м | 50м | 100м |
|--------|-----|-----|------|------|-------|
| 1 Вт | ✅ | ✅ | ⚠️ | ❌ | ❌ |
| 10 Вт | ✅ | ✅ | ✅ | ⚠️ | ❌ |
| 100 Вт | ✅ | ✅ | ✅ | ✅ | ⚠️ |
| 1000 Вт | ✅ | ✅ | ✅ | ✅ | ✅ |
| 10 кВт | ✅ | ✅ | ✅ | ✅ | ✅ |

## 4. Контрбатарейная разведка

### 4.1 Обнаружение артиллерийских позиций

DAS-система обнаруживает:
1. **Выстрел** — импульс в точке орудия (мощность 0.85-1.0, 0.5-3 Гц)
2. **Полёт снаряда** — акустическая тень (если пролетает рядом)
3. **Разрыв** — мощный импульс (0.9-1.0, 0.1-15 Гц)

По разнице времён:
```
Δt = t_разрыва - t_выстрела
D = V_снаряда × Δt (приблизительно)
```

### 4.2 Точность

| Параметр | Значение |
|----------|----------|
| Обнаружение выстрела | 95-99% на расстоянии до 2 км от кабеля |
| Локализация орудия | ±50 м (требует калибровки) |
| Обнаружение разрыва | 99% на расстоянии до 5 км |
| Локализация разрыва | ±15 м |
| Время обнаружения | < 1 секунда |
| Передача координат | < 3 секунд |

### 4.3 Автоматическое целеуказание

```
[Обнаружен выстрел] → [Классификация: 152мм гаубица]
                     → [Расчёт координат]
                     → [Отправка на огневую позицию]
                     → [Контрбатарейный удар через 30-60 сек]
```

## 5. Мониторинг линий снабжения

### 5.1 Контроль дорог

Волокно, проложенное вдоль дороги, фиксирует:
- **Направление движения** (доплеровский сдвиг)
- **Скорость** (время прохождения между точками)
- **Тип транспорта** (колёсный/гусеничный)
- **Количество единиц** (подсчёт импульсов)

### 5.2 Пример отчёта

```
=== TFN ISR REPORT ===
Время: 2026-04-18 03:45:12
Сегмент: Road monitoring, 1200-3500m

DETECTED:
  03:42:15 — Колёсная техника ×3, позиция 1450м, скорость ~35 км/ч, направление Ю→С
  03:43:30 — Колёсная техника ×3, позиция 2800м, скорость ~30 км/ч, направление Ю→С
  03:44:50 — Гусеничная техника ×1, позиция 1350м, скорость ~15 км/ч, направление Ю→С

ASSESSMENT: Колонна из 4+ единиц (3 колёсных + 1 гусеничная)
           Движение на север по маршруту supply-road-7
           Возможная переброска резервов

RECOMMENDATION: Наблюдение. Подготовка огневого задания.
```

## 8. Интеграция с картой (Tactical Map)

### 8.1 Формат событий

```json
{
  "event_id": "evt_20260418_034512_001",
  "timestamp": "2026-04-18T03:45:12Z",
  "type": "target_detected",
  "source": "DAS",
  "position": {
    "fiber_m": 1450,
    "lat": 48.5123,
    "lon": 36.3456
  },
  "classification": {
    "target": "wheeled_vehicle",
    "confidence": 0.87,
    "count": 3,
    "speed_kmh": 35,
    "direction": "N"
  },
  "threat_level": "medium",
  "action": "track",
  "alert_sent": true
}
```

### 8.2 Визуализация

Каждое событие отображается на тактической карте:
- **Зелёный** — тихо
- **Жёлтый** — пешеход, копание
- **Оранжевый** — колёсная техника, БПЛА
- **Красный** — гусеничная техника, РЭБ
- **Мигающий красный** — артиллерия, взрыв

## 9. Защита от контрмер

### 9.1 Противник не может обнаружить TFN

| Метод обнаружения | Эффективность против TFN |
|-------------------|--------------------------|
| РТР (пеленгация) | ❌ Нет излучения |
| Тепловизор | ❌ Нет нагрева |
| Металлоискатель | ❌ Стекло, не металл |
| Визуально | ⚠️ Только если наткнётся |
| GPR (георадар) | ⚠️ Возможно на глубине < 20 см |

### 9.2 Противник может попытаться

| Действие | Эффективность | Контрмера TFN |
|----------|---------------|---------------|
| Обрыв кабеля артиллерией | Частично | Mesh-резервирование, автопереключение |
| Перехват оптического сигнала | Нужен физический доступ | Шифрование WireGuard |
| Имитация вибраций | Возможна дезинформация | ML-модель фильтрует |
| Обнаружение лазера φ-OTDR | Нет, весь свет внутри волокна | — |
