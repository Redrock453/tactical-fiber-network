# 🕸️ Tactical Fiber Network (TFN) — Project "SpiderLink"

**Tactical Fiber Network (TFN)** — це open-source концепція розгортання прихованих мереж зв'язку та сенсорних полів на базі відпрацьованого мікро-оптоволокна БПЛА.

---

## 🎯 Мета проекту

Перетворення "одноразового" кабелю керування дронами на стійку до РЕБ та РЕР мережеву інфраструктуру.

---

## 🚀 Ключові інновації

### 1. Zero-Emission Communication
Повна відсутність радіовипромінювання. Мережа невидима для засобів РЕР противника.

### 2. Mesh-Redundancy
Використання множини ліній від різних вильотів для створення надлишкової мережі. Немає єдиної точки відмови.

### 3. DAS (Distributed Acoustic Sensing)
Використання волокна як датчика руху. Кожна нитка "чує" кроки та техніку в радіусі дії.

---

## 4. Розвідувальні можливості (ISR Capabilities)

Tactical Fiber Network (TFN) перетворює звичайний оптоволоконний кабель на потужну **пасивну розвідувально-охоронну систему** класу Distributed Acoustic Sensing (DAS). Завдяки технології Rayleigh backscattering весь кабель стає тисячами віртуальних сенсорів, що працюють у реальному часі без жодного радіовипромінювання.

### 4.1 Принцип роботи DAS-розвідки в TFN

- Лазерний імпульс поширюється по волокну.
- Аналіз зворотного розсіювання (backscatter) дозволяє фіксувати вібрації та акустичні хвилі з точністю до 1–10 метрів.
- Ефективна дальність: **10–40 км** по одному волокну (залежно від лазера та посилювачів).
- Чутливість: виявляє кроки людини, рух техніки, роботу двигунів, копання, постріли, вибухи та навіть низькочастотні вібрації від БПЛА.

**Ключова перевага** — **Zero-Emission ISR**. Система повністю пасивна і невиявна для ворожих РЕБ-засобів та пеленгаторів.

### 4.2 Тактичні сценарії застосування

| Сценарій | Що виявляє TFN-DAS | Тактичний ефект |
|----------|-------------------|-----------------|
| Охорона бази/позиції/складу | Кроки, техніка, копання, дрони-камікадзе | Раннє попередження без камер і радарів |
| Моніторинг "сірої зони" | Рух ворожих ДРГ, БМП, танків | Контроль руху в умовах тотального РЕБ |
| Моніторинг тилових комунікацій | Артилерійські/дронові удари по кабелю | Автоматичне визначення координат ворожих позицій |
| Trophy Intelligence | Підключення до ворожого оптоволокна | Слухання ворожої активності навіть після обриву |
| Розвідка артилерійських позицій | Вібрації від пострілів і падіння снарядів | Ціль наведення для контрбатарейної боротьби |

### 4.3 Режим "Trophy Intelligence"

Одна з найбільш перспективних можливостей TFN:
- Якщо противник розгортає власну оптоволоконну мережу — TFN може **підключитися** до неї.
- Через аналіз backscatter-сигналу оператор отримує дані про рух і активність вздовж ворожого кабелю.
- Навіть обірваний ворожий кабель продовжує працювати як сенсор на відстані до кількох кілометрів.

Це дає змогу перетворювати ворожу інфраструктуру на джерело розвідданих.

### 4.4 Переваги над традиційними засобами розвідки

- **Невиявність** — 100% відсутність радіосигналу.
- **Безперервність** — працює 24/7 навіть у повному РЕБ.
- **Масштабованість** — один кабель = десятки кілометрів фронту.
- **Резервування** — mesh-архітектура + DAS працюють навіть при пошкодженнях.
- **Вартість** — значно дешевше, ніж мережа камер + радарів + РЕБ-захисту.

**TFN робить неможливе раніше — пасивну, стійку до РЕБ розвідку на великій глибині та ширині фронту.**

## 📖 Повна документація

### DAS (Distributed Acoustic Sensing)

Оптоволокно чутливе до мікро-деформацій. Ми використовуємо метод фазочутливої рефлектометрії (φ-OTDR) для перетворення прокладеного кабелю на розподілений датчик.

#### Варіанти застосування:

1. **Периметральний захист:** Детекція кроків піхоти (локалізація до 5 метрів)
2. **Трофейна розвідка:** Підключення до обривків кабелю противника для віддал��ного прослуховування активності на їх позиціях (без заходу на територію)
3. **Контроль цілісності:** Автоматичне визначення місця обриву кабелю артилерією

#### Технічна реалізація:

- Використання когерентних лазерних джерел
- Аналіз зворотного релеївського розсіювання
- Класифікація цілей через ML-моделі (кроки, колісна техніка, гусенична техніка)

#### Як працює трофейна розвідка:

1. Підключаємо **інтеррогатор** (прилад, який посилає імпульси і слухає ехо) до кінця кабелю
2. Навіть якщо кабель обірваний на тій стороні — отримуємо відбитий сигнал від кожної мікронерівності всередині скла
3. Будь-який рух поряд з кабелем на стороні противника змінює фазу цього відображення
4. **Результат:** Бачимо на моніторі графік активності — "4000 метрів від нас — рух техніки"

---

### Розгортання вузла TFN

#### Склад вузла (Node):

1. **L3 Switch/Router** (з підтримкою SFP або через медіаконвертери)
2. **Автономне живлення:** LiFePO4 збірка на 12/24V + інвертор/DC-DC
3. **Механічні сплайси:** Набір для холодного з'єднання волокна (без зварки)

#### Алгоритм розгортання:

1. Збір даних про траєкторії вильотів БПЛА
2. Пошук та маркування точок входу/виходу кабелю
3. Встановлення "Trench-Nodes" у захищених точках
4. Програмне налаштування Mesh-інтерфейсів для автоматичного перемикання при обривах

## 🇬🇧 English

### About

SpiderLink (TFN) is an open-source concept for deploying hidden communication networks and sensor fields using spent micro-optical fiber from FPV drones.

### Key Features

- **Zero-RF Emission:** Completely invisible to enemy EW/ECM systems
- **Mesh Topology:** No single point of failure
- **DAS Sensing:** Each fiber strand acts as a distributed acoustic sensor
- **Zero Cost:** Uses spent fiber from drone operations

### Advantages

| Parameter | RF-link | **SpiderLink TFN** |
|-----------|--------|-------------------|
| Stealth | Detected by ECM in seconds | **0% RF emission** |
| EWimmunity | Fully jammed | **Absolute immunity** |
| Operator safety | 0–3 km from front | **20+ km rear** |
| Cable cost | Separate expense | **$0 (spent)** |

### Use Cases

- Secure command links in EW-saturated environments
- Perimeter intrusion detection
- Trophy intelligence on enemy fiber lines
- Damage assessment and break localization

### Technical Stack

- **Physics:** Single-mode fiber (G.657.A2), SFP modules (1310/1550nm)
- **Routing:** B.A.T.M.A.N. adv / OLSR
- **Analytics:** Python scripts for attenuation and vibration analysis

| SFP modules | $30–50 |
| Optical switch | $50–100 |
| **Total per node** | **$140–250** |
| **Mesh 5 nodes** | **$700–1250** |
