## 2. MCU Платформи для Signal Acquisition

### 2.1 ESP32-S3 (Espressif)

| Параметр | Значення |
|---|---|
| Ціна Amazon (DevKitC-1 N8R8) | $12-15 |
| Ціна AliExpress | $3-5 |
| Ядро | Dual-core Xtensa LX7, 240 MHz |
| ADC | 12-bit, 2 ADCs, до 20 каналів |
| Max ADC sample rate | ~2 MHz (теоретично), реально ~83 kHz (I2S mode) |
| I2S | Так, може використовуватись для ADC DMA |
| WiFi | 802.11 b/g/n |
| Bluetooth | BLE 5.0 |
| Flash/PSRAM | 8MB Flash / 8MB PSRAM (N8R8) |
| SPI | Так (для зовнішнього ADC) |
| Напруга | 3.3V |

**ADC якість ESP32-S3:** Вбудований ADC — 12-bit, нелінійний, з шумом. Непридатний для точних вимірів. **Рекомендується зовнішній ADC** (ADS1256 через SPI).

**I2S ADC mode:** ESP32 може семплити через I2S peripheral з DMA на частотах до ~80 kHz. Це дозволяє безперервний потік даних без участі CPU.

**Вердикт:** Найкращий вибір для MVP — потужний процесор, WiFi для передачі даних, I2S для швидкого ADC, дешевий.

### 2.2 ESP32 з I2S ADC

Оригінальний ESP32 (не S3) також має I2S ADC mode:

| Параметр | Значення |
|---|---|
| I2S ADC sample rate | До 80 kHz підтверджено |
| ADC розрядність | 12-bit (але реально ~10-bit through I2S через noise) |
| DMA буфер | До 1024 семплів |

**Чи може ESP32 семплити на 10 kHz+?** ТАК. Через I2S ADC mode — легко. Але якість вбудованого ADC обмежена. Для кваліфікації сигналу краще зовнішній ADC.

## 4. Signal Conditioning

### 4.1 LM358 Op-Amp Circuit для Piezo

Найпростіше рішення для підсилення piezo сигналу:

```
                    +3.3V
                      |
                     [R1] 1MΩ
                      |
Piezo+ ----||---------+-----------> до ADC
        C_piezo        |
                      [R2] 100kΩ
                      |
                     GND

LM358 як voltage follower або non-inverting amplifier:
  - Gain = 1 + Rf/Rin
  - Для сейсміки: Gain = 10-100
  - Bandwidth: достатньо для 0.1-1000 Hz
```

**Ціна LM358:** $0.10-0.30 (AliExpress, 10 шт за $1-3)

**Нюанси:**
- LM358 — single-supply op-amp, працює від 3-32V
- Вхідний bias current може бути проблемою для piezo (потрібен FET-input op-amp краще)
- Альтернатива: TL072, MCP6002 (FET input, rail-to-rail)

### 4.2 Charge Amplifier для Piezo

Charge amplifier — оптимальна схема для piezo сенсорів, перетворює charge (заряд) piezo у напругу:

```
         Cf
Piezo --||---+---- Vout
              |
             |-\
         Rf  |  \
        --[]--+   )--+-- GND
        |        /
        |       |+/
        |       |
        +---GND-+
```

- Cf = 1-10 nF (визначає чутливість)
- Rf = 1-10 MΩ (визначає lower cutoff frequency: f_L = 1/(2π·Rf·Cf))
- При Rf=10MΩ, Cf=10nF: f_L = 1.6 Hz (ідеально для сейсміки)
- Верхня частота обмежена bandwidth op-amp

**Рекомендація:** MCP6002 (rail-to-rail, FET input) або TLV2462.

### 4.3 Band-Pass Filter (1-200 Hz для сейсміки)

```
Вхід ---[R1]---+---[R2]---+--- Вихід
               |           |
              C1|         C2|
               |           |
              GND         GND

High-pass (C1, R1): f_H = 1/(2π·R1·C1) = 1 Hz → R1=160kΩ, C1=1µF
Low-pass (R2, C2):  f_L = 1/(2π·R2·C2) = 200 Hz → R2=8kΩ, C2=100nF
```

**Для різних цілей:**
| Ціль | Смуга частот | Компоненти |
|---|---|---|
| Кроки пішохода | 1-50 Hz | R=320kΩ/C=0.5µF, R=32kΩ/C=100nF |
| Техніка/автомобілі | 2-200 Hz | R=160kΩ/C=0.5µF, R=8kΩ/C=100nF |
| Загальний acoustic | 10-2000 Hz | R=16kΩ/C=1µF, R=800Ω/C=100nF |

**Ціна компонентів:** $0.50-1.00 за повний фільтр (резистори + конденсатори).

### 5.3 Як прикріпити Piezo до ґрунту/кабелю

**Найкращі методи coupling:**

1. **Грунт (ground contact):**
   - Приклеїти piezo disc до металевої пластини (алюміній 2-3mm)
   - Пластину втопити в ґрунт, придавити каменем
   - Альтернатива: заливка piezo в epoxy з плоским дном
   - **Ключ:** максимальна площа контакту з ґрунтом

2. **Кабель (fiber/cable monitoring):**
   - Piezo прикріплений до cable jacket за допомогою hose clamp
   - Передавання вібрації кабелю на piezo
   - Звучить як "true DAS" — але з discrete sensors

3. **Акустичний coupling:**
   - Використовувати acoustic coupling gel (як для УЗД)
   - Або двосторонню скотч-стрічку (3M VHB)
   - Silicone adhesive також працює

## 10. Висновки та Рекомендації

### Для MVP Pseudo-DAS рекомендується Варіант B ($141):

1. **ADS1256** — ключовий компонент. 24-bit при 30 kSPS дозволяє повноцінний frequency analysis
2. **ESP32-S3** — потужний MCU з WiFi для data streaming, I2S для potential built-in ADC backup
3. **Piezo disc 27mm** — дешевий, замінний, доступний. 4-8 штук на один ADS1256
4. **AD620 modules** — швидке prototyping signal conditioning
5. **IP67 корпус** — для outdoor deployment

### Наступні кроки:
1. Зібрати один вузол Варіанту B
2. Написати firmware для ESP32-S3 (SPI → ADS1256 → буфер → WiFi)
3. Протестувати з piezo на різних поверхнях (ґрунт, асфальт, кабель)
4. Виміряти реальний SNR та дальність детектування
5. Збільшити мережу до 4-8 вузлів
6. Додати ML-модель для класифікації подій

### Ключові формули для розрахунку:

```
Nyquist frequency = Sample Rate / 2
  ADS1115: 860/2 = 430 Hz max
  ADS1256: 30000/2 = 15000 Hz max

SNR (dB) = 6.02 × N + 1.76
  12-bit: 74 dB
  16-bit: 98 dB
  24-bit: 146 dB

Seismic wavelength = Wave velocity / Frequency
  Soil (P-wave ~300 m/s) at 10 Hz: λ = 30 m
  Soil (P-wave ~300 m/s) at 50 Hz: λ = 6 m
  Soil (P-wave ~300 m/s) at 200 Hz: λ = 1.5 m

Sensor spacing recommendation:
  For spatial aliasing prevention: Δx < λ_min / 2
  For 50 Hz: Δx < 3 m
  For 200 Hz: Δx < 0.75 m
```
