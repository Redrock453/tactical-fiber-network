## Quick Start (3 команды)

```bash
git clone https://github.com/Redrock453/tactical-fiber-network.git
cd tactical-fiber-network
pip install -r requirements.txt

# Запустить демо DAS-симулятора (обнаружение целей)
python -m simulation.das_simulator

# Запустить демо mesh-сети (развертывание + артобстрел + восстановление)
python -m simulation.mesh_simulator

# Запустить калькулятор оптического бюджета
python -m calculator.fiber_budget

# Запустить Streamlit дашборд
streamlit run web/dashboard.py
```

## Что делает TFN

Превращает **«одноразовый» кабель от FPV-дронов** в устойчивую к РЭБ сетевую инфраструктуру:

```
                   ┌─────────────┐
                   │  Master Node │  (тыл, ML-анализ)
                   │  + φ-OTDR    │
                   └──────┬──────┘
                          │ fiber
          ┌───────────────┼───────────────┐
          │               │               │
    ┌─────┴─────┐   ┌─────┴─────┐   ┌─────┴─────┐
    │  Node A   │   │  Node B   │   │  Node C   │
    │  (окоп)   │───│  (реле)   │───│  (ДРГ)    │
    └───────────┘   └───────────┘   └───────────┘
         │               │               │
    [DAS: шаги]    [DAS: техника]  [RF: РЭБ]
```

### 4 ключевых инновации

| # | Инновация | Что даёт |
|---|-----------|----------|
| 1 | **Zero-Emission Communication** | 0% радиоизлучения — невидим для РЭР/РТР |
| 2 | **Mesh-Redundancy** | Нет единой точки отказа — работает при обрывах |
| 3 | **DAS Sensing** | Волокно = распределённый акустический датчик (кроки, техника, артснаряды) |
| 4 | **Trophy Intelligence** | Подключение к вражеским линиям — их кабель становится нашим сенсором |

## Симуляторы (запустить без железа)

### DAS-симулятор — обнаружение целей

```bash
python -m simulation.das_simulator
```

Обнаруживает 11 типов целей по вибрациям волокна:
Пешеходы, техника (колёсная/гусеничная), артиллерия, взрывы, БПЛА, РЭБ, копание

### Mesh-симулятор — развёртывание и обрывы

```bash
python -m simulation.mesh_simulator
```

Развёртывает mesh-сеть из 8 узлов, симулирует артобстрел, пересчитывает маршруты.

### RF-детектор — пассивное обнаружение РЭБ

```bash
python -m simulation.rf_detector
```

Моделирует обнаружение РЭБ-станций, радаров, FPV-каналов через эффект Керра.

### Калькулятор оптического бюджета

```bash
python -m calculator.fiber_budget
```

Сравнивает 6 сценариев (идеальный, полевой, отработанное волокно, экстренный).

### Оценка потерь сплайсов

```bash
python -m calculator.splice_loss_estimator
```

Сравнивает все методы соединения + Monte Carlo симуляция надёжности.

## Полевое соединение волокна (без сварки!)

На фронте **нет сварочного аппарата**. Но есть:

| Метод | Время | Потери | Цена | Под огнём |
|-------|-------|--------|------|-----------|
| Механический сплайс (FMS-01) | **30 сек** | 0.1-0.2 дБ | $10 | Да |
| Быстрый коннектор (SC) | **60 сек** | 0.2-0.3 дБ | $3 | Да |
| Сварка (fusion) | 5-15 мин | 0.02 дБ | $2000+ | Нет |

Минимальный набор для работы: **стриппер ($5) + сколом ($15) + 5 сплайсов ($50) = $70**

Подробнее: [docs/FIELD_SPLICING_GUIDE.md](docs/FIELD_SPLICING_GUIDE.md)

## Разведка через TFN

### DAS-разведка (акустическая)

| Цель | Дальность от кабеля | Локализация | Классификация |
|------|---------------------|-------------|---------------|
| Пешеход | 3-10 м | ±2 м | 75-85% |
| БТР/БМП | 15-50 м | ±5 м | 85-95% |
| Танк | 20-80 м | ±5 м | 90-97% |
| Артвыстрел | 50-500 м | ±10 м | 95-99% |
| Копание | 5-15 м | ±3 м | 70-85% |

### RF-Opto разведка (пассивная)

| Источник | Дальность обнаружения | Вероятность |
|----------|----------------------|-------------|
| РЭБ-станция (1кВт) | 5-50 м | 90-99% |
| FPV-пульт | 1-15 м | 40-70% |
| Радар ПВО | 50-500 м | 95-99% |

### Трофейная разведка

Подключение к обрывку вражеского кабеля за 3-5 минут → чтение активности на позициях противника.

Подробнее:
- [docs/ISR_INTELLIGENCE_GUIDE.md](docs/ISR_INTELLIGENCE_GUIDE.md) — полная разведывательная система
- [docs/TROPHY_INTELLIGENCE.md](docs/TROPHY_INTELLIGENCE.md) — трофейная разведка

## Технический стек

- **Физика:** Single-mode fiber (G.657.A2), SFP-модули (1310/1550nm)
- **Маршрутизация:** B.A.T.M.A.N. adv / OLSR
- **Обработка:** Python (NumPy, FFT), ML-классификация
- **Визуализация:** Streamlit
- **Связь:** FastAPI (SSE), Telegram Bot
- **Безопасность:** WireGuard (ChaCha20-Poly1305)
- **Развертывание:** Docker, DigitalOcean
