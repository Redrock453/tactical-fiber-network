# 🕸️ Tactical Fiber Network (TFN) — Project SpiderLink

**Tactical Fiber Network (TFN)** — це open-source концепція розгортання прихованих мереж зв'язку та сенсорних полів на базі відпрацьованого мікро-оптоволокна БПЛА.

---

## 🎯 Мета проекту

Перетворення "одноразового" кабелю керування дронами на стійку до РЕБ та РЕР мережеву інфраструктуру.

---

## 🚀 Ключові інновації

1. **Zero-Emission Communication** — повна відсутність радіовипромінювання
2. **Mesh-Redundancy** — надлишкова мережа без єдиної точки відмови
3. **DAS** — використання волокна як датчика руху
4. **RF-Opto Hybrid** — волокно як пасивна антена
